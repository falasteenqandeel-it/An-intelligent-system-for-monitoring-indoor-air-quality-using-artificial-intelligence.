# An-intelligent-system-for-monitoring-indoor-air-quality-using-artificial-intelligence.
An intelligent system for monitoring indoor air quality using artificial intelligence.
Our project is an intelligent indoor air quality monitoring system using artificial intelligence. This system continuously measures temperature, humidity, and carbon dioxide levels in real time, then analyzes and displays this data in a clear and easy-to-understand format.

We chose this project because we believe that human health begins with the surrounding environment. We also noticed a lack of awareness about indoor air quality, despite its significant impact on concentration and overall health. Therefore, we wanted to offer a smart, simple, and practical solution that helps improve daily life.
